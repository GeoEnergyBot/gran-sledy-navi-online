# Запуск «Грань: Следы Нави» онлайн на Railway

## Архитектура первой онлайн-альфы

- один новый GitHub-репозиторий;
- один новый Railway-проект;
- один постоянный Railway Volume, подключённый как `/data`;
- один отдельный Telegram-бот;
- сервер и клиент работают на одном HTTPS-домене;
- Supabase и Vercel на этом этапе не используются.

Это сохраняет весь текущий функционал: общий мир, SSE-обновления, рынок, Круги и файл состояния. Vercel нельзя использовать как прямую замену этому серверу без переработки хранилища и real-time слоя.

## Переменные Railway

```env
NODE_ENV=production
HOST=0.0.0.0
DATA_DIR=/data
DEMO_MODE=1
WORLD_SEED=СЛУЧАЙНЫЙ_СЕКРЕТ
MARKET_COMMISSION_PERCENT=7
ENABLE_CRYPTO=0
```

`PORT` не задавайте: Railway передаёт его приложению автоматически.

После проверки через браузер добавьте:

```env
BOT_TOKEN=ТОКЕН_НОВОГО_TELEGRAM_БОТА
DEMO_MODE=0
```

## Проверка

- `https://ВАШ-ДОМЕН/health` должно вернуть JSON с `"ok": true`.
- Главная страница должна открыть карту.
- В двух разных браузерах в режиме `DEMO_MODE=1` создаются разные тестовые игроки, но они видят общий мир и рынок.
