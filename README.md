# Грань: Следы Нави — Multiplayer Alpha

Рабочий вертикальный срез многопользовательской геолокационной RPG. Все игроки подключаются к одному серверному миру, видят одинаковые события на реальной карте, совместно закрывают разломы, добывают материалы, создают амулеты и торгуют предметами.

## Что реализовано

- единый детерминированный мир на реальной карте;
- общие существа, следы, артефактные тайники, места силы и разломы;
- персональное завершение обычных событий и общий прогресс разломов;
- серверная геопроверка и базовая защита от телепортов;
- три мини-игры: преследование, поиск сигнала и ритуал;
- выбор исхода: успокоить, исследовать или изгнать;
- 12 существ славянской мифологии и журнал знаний;
- материалы, расходники, амулеты и уникальные артефакты с историей владельцев;
- ремесло и четыре профессии;
- Убежище, экипировка и улучшения;
- ежедневные задания;
- многопользовательские Круги до 50 участников;
- рынок игроков: выставление, покупка, отмена лотов и комиссия разработчика;
- Server-Sent Events для обновлений рынка, разломов и Кругов;
- Telegram Mini App авторизация с проверкой `initData` при наличии `BOT_TOKEN`;
- отключённый по умолчанию криптовалютный слой, подготовленный для отдельного TON-адаптера;
- PWA-оболочка и резервная карта, если Leaflet/CDN недоступны.

## Быстрый запуск

Нужен Node.js 20 или новее. В проекте нет внешних npm-зависимостей.

```bash
cd gran_nav_mmo
npm start
```

Откройте:

```text
http://localhost:8080
```

Для разработки:

```bash
npm run dev
```

Автоматическая проверка основных механик:

```bash
npm test
```

## Настройка

Переменные окружения перечислены в `.env.example`.

Для локального запуска используется `DEMO_MODE=1`. В продакшене:

```bash
DEMO_MODE=0 BOT_TOKEN=ваш_токен WORLD_SEED=секретный_случайный_ключ node server.mjs
```

При `DEMO_MODE=0` сервер принимает только Telegram Mini App `initData`, подпись которого проверяется на сервере.

## Данные

Состояние хранится в `data/state.json`. Это удобно для тестов и небольшой закрытой альфы. Для публичного запуска JSON-хранилище следует заменить PostgreSQL/PostGIS и Redis, сохранив текущие API-контракты.

Чтобы начать новый локальный мир, остановите сервер и удалите:

```text
data/state.json
```

## Реальная карта

Leaflet и тайлы OpenStreetMap загружаются через CDN. При отсутствии доступа к CDN клиент автоматически показывает резервную стилизованную координатную карту, а механики продолжают работать.

## Криптовалюта

Внутренняя валюта `Знаки Грани` сейчас офчейн. `ENABLE_CRYPTO=1` только меняет статус готовности интеграции — реальные операции не выполняются. Подключать TON следует отдельным лицензированным адаптером после аудита безопасности и юридической проверки. Обычные материалы рекомендуется оставить на сервере, а в блокчейн выводить только редкие сертифицированные артефакты и торговую валюту.

## Важное ограничение

Это функциональная multiplayer alpha, а не готовая к миллионам пользователей production-MMO. Перед коммерческим запуском обязательны PostgreSQL/PostGIS, Redis, горизонтальное масштабирование, полноценная модерация, мониторинг, антифрод, резервное копирование и аудит экономики/смарт-контрактов.

## Онлайн-развёртывание Railway

Для текущей архитектуры используйте Railway с постоянным Volume, смонтированным в `/data`, и переменной `DATA_DIR=/data`. Подробная инструкция находится в `DEPLOY_RAILWAY_RU.md`.

Проверка состояния сервера:

```text
GET /health
```
